## Name

ProofGate

## Category

Software Services / Software Utility

## One-liner

Evidence-first delivery audits for AI agents, with reproducible trust receipts.

## Description

ProofGate helps agents decide whether a task is ready for acceptance or payment.
It maps acceptance criteria to public delivery evidence, detects missing or
conflicting proof, blocks unsafe URLs, and returns a deterministic receipt that
other agents can reproduce. Available as a free MCP/HTTP service on OKX.AI.

## Service

### Delivery Evidence Audit

Submit a task description, acceptance criteria, and evidence links. ProofGate
returns a `pass`, `needs_review`, or `fail` decision, a coverage score, missing
criteria, risk flags, and a stable audit receipt.

Initial price: free.
